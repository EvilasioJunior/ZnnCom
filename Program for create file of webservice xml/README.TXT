======================== Descrição do Aplicativo ========================


Ele cria um arquivo xml com nome "WebserviceXML.xml" contendo a representação de uma lista de web services. 

======================== Funcionalidades ========================


- Adicionar: Adiciona web service com as informações que estão nos campos do formulário. Caso o campo não seja preenchido, adiciona a informação vazia. Se o arquivo "WebserviceXML.xml" ainda não existir ele cria o arquivo.
- Limpar Formulário: Limpa os campos do formulário
- Apagar arquivo atual: Deleta arquivo WebserviceXML.xml
- Cancelar: Fecha aplicação
